Votre navigateur est un Internet Explorer de version antérieure à 9 qui n'est pas supportée par l'application Alterconsos.

Vous avez plusieurs options possibles :

  • Rechercher sur le Web le site de Microsoft qui indiquera comment télécharger et installer une version 9 ou postérieure. Selon votre version de Windows ceci n'est pas toujours possible ;
  • Installer (en plus d'Internet Explorer) un autre navigateur. Les deux qui fonctionnent le mieux sont :
    • Google Chrome : Téléchargement ici
    • Firefox (conseillé pour Windows XP): Téléchargement ici
    • Internet Explorer 9 (Windows Vista et 7): Téléchargement ici

A bientôt sur l'application Alterconsos.
